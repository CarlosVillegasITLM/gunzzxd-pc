# GunZzxD-PC

Repositorio de respaldo y gestión de datos de Pokémon de múltiples generaciones.

## Descripción

Este proyecto contiene datos y archivos de guardado de Pokémon desde diferentes generaciones del juego, incluyendo archivos de Pokémon individuales (.pk7) y archivos de guardado completos. Se utiliza como respaldo versionado del progreso en diferentes juegos de Pokémon.

## Estructura del Proyecto

El proyecto está organizado por generaciones de Pokémon:

- **Gen I** - Primera generación (Rojo/Azul/Amarillo)
- **Gen III - RFVH** - Tercera generación (Rojo Fuego/Verde Hoja)
- **Gen III - RZE** - Tercera generación (Rubí/Zafiro/Esmeralda)
- **Gen VI - ORZA** - Sexta generación (Oro/Rosa/Diamante)
- **Gen VII - USUM** - Séptima generación (Ultra Sol/Ultra Luna)
- **Gen VIII - SWSH** - Octava generación (Espada/Escudo)

Cada carpeta contiene:
- Archivos de Pokémon individuales (formato `.pk7`)
- Carpetas de archivos de guardado

## Características

- 📦 Respaldos versionados de datos de Pokémon
- 📁 Organización por generación de juegos
- 💾 Archivos de guardado completos
- 🔄 Historial de cambios mediante Git

## Uso

Este repositorio sirve como:
- Backup de progreso en diferentes juegos de Pokémon
- Histórico de cambios y actualizaciones
- Almacenamiento de datos de Pokémon capturados

## Historial

El proyecto comenzó como un respaldo de la tarjeta SanDisk MicroSD 16GB y se ha mantenido con actualizaciones periódicas del progreso en diferentes generaciones de Pokémon.
